# D8starter
Drupal 8 starter theme, bootstrap 3, gulp, sass and font awesome included.

What do you need?

1. Node.js
2. Gulp

How to use?

1. Go to your dir and simply clone repo in console > git clone https://github.com/kwegiel/d8starter.git .
2. Type > npm install
3. You're ready to go > gulp watch
